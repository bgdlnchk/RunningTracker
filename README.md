# RunningTracker

<h2>Description</h2>

A running tracker Android application, build with the newest Android Development technologies.

<h2>Features</h2>

- Start a run;
- Check previous runs information;
- See your route on the map;

<h2>Technologies</h2>

- Android Studio;
- Koltlin;
- MVVM;
- Room;
- DI;
- Fragments;
